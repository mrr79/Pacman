\doxysubsection{Detailed Description}
Clase que representa un fantasma 3 en el juego. Este fantasma es el fantasa de color celeste y se mueve ppersiguiendo al pacman o al poder usando el algoritmo A$\ast$. 

Esta clase hereda de QObject y QGraphics\+Pixmap\+Item, y representa un fantasma en el juego.
