\doxysubsection{Detailed Description}
Clase que representa un fantasma 2 en el juego. Este fantasma es el fantasa de color rosado y se mueve persiguiendo al pacman o al poder usando el algoritmo \mbox{\hyperlink{classBacktracking}{Backtracking}}. 

Esta clase hereda de QObject y QGraphics\+Pixmap\+Item, y representa un fantasma en el juego.
