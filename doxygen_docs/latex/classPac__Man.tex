Inicia el servidor de socket para recibir mensajes entrantes. 

Esta función crea un socket, configura la dirección y el puerto, y escucha conexiones entrantes. Una vez que se establece una conexión, se procesan los mensajes recibidos y se realiza la acción correspondiente. Los mensajes deben tener el formato \char`\"{}giro\+\_\+x,giro\+\_\+y,giro\+\_\+z\char`\"{}, donde los valores de giro\+\_\+x, giro\+\_\+y y giro\+\_\+z son números en punto flotante. Basándose en los valores recibidos, se establece la dirección de movimiento del Pac-\/\+Man. También se actualizan valores del juego como el puntaje. Finalmente, se envía una respuesta al cliente y se cierran los sockets.
