Detailed Description

Clase que representa un fantasma 2 en el juego. Este fantasma es el fantasa de color rosado y se mueve persiguiendo al pacman o al poder usando el algoritmo Backtracking.

Esta clase hereda de QObject y QGraphicsPixmapItem, y representa un fantasma en el juego.
